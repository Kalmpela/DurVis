## DurVis
Durvis is a Bakkesmod plugin to visualize and diagnose Durazno^2.

### Requirements
* [Rocket League](https://www.rocketleague.com/)
* [Bakkesmod](http://bakkesmod.com/)
* [Durazno^2](http://kalma.kapsi.fi/rldurazno/)

### To compile
* [Bakkesmod SDK](https://github.com/Bakkes/BakkesModSDK)
* [Boost libraries](https://www.boost.org/)
